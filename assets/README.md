# assets/

Drop your branding here:

- `channel_cover.jpg` — 1000×1000 default background used by the cover generator
  when a track has no artwork (optional; a gradient is generated if absent).
- `watermark_logo.png` — transparent PNG logo, stamped bottom-right at ~15%
  opacity on every cover (optional; skipped if absent).

Both are optional — the bot renders clean generated covers without them.
